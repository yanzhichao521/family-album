# FamilyAlbum 中转即焚多人相册

一个适合家庭使用的中转即焚多人相册应用，支持图片和视频上传、成员邀请、文件自动过期等功能。

## 功能特点

- 📷 支持图片和视频上传
- 👥 邀请码加入相册空间
- 💾 文件下载后自动标记
- ⏰ 30天自动过期
- 👍 点赞和评论功能
- 🔒 每个空间最多10人

## 📦 快速开始（本地测试）

### 前置要求

- Python 3.10+
- Node.js 18+

### 方法一：一键启动（Windows）

1. 双击运行 `start.bat`
2. 等待自动安装依赖和启动服务
3. 浏览器访问 http://localhost:3000

### 方法二：手动启动

**第一步：启动后端**

```bash
cd backend
pip install -r requirements.txt
python init_db.py
python main.py
```

**第二步：启动前端（新开一个终端）**

```bash
cd frontend
npm install
npm run dev
```

**第三步：访问应用**

打开浏览器访问 http://localhost:3000

## 📱 如何邀请朋友使用

### 1. 创建相册空间

1. 登录应用
2. 点击「创建空间」
3. 输入空间名称（如「家庭相册」）

### 2. 获取邀请码

1. 进入创建的空间
2. 点击「邀请」按钮
3. 复制邀请码

### 3. 分享给朋友

把**访问地址**和**邀请码**一起发给朋友：
- 地址：`https://your-app.onrender.com`（云平台地址）
- 邀请码：`ABCD1234`

### 4. 朋友加入

1. 朋友打开地址
2. 注册/登录账号
3. 点击「加入空间」
4. 输入邀请码
5. 开始上传和查看照片！

## ❓ 常见问题

### Q: 免费云平台稳定吗？
A: Render、Railway 等平台非常稳定，24小时在线，适合长期使用。

### Q: 免费额度够用吗？
A: Render 提供 750小时/月，相当于 24小时×31天，完全够用。

### Q: 邀请码无法加入？
A: 确保邀请码输入正确，检查是否有多余的空格。

### Q: 文件上传失败？
A: 检查文件大小，确保有足够的存储空间。

### Q: 如何备份数据？
A: 建议定期导出数据库文件 `backend/familyalbum.db`。
