## 方案一：花生壳（最推荐）

### 为什么推荐花生壳？
- ✅ 国内服务商，访问速度快
- ✅ 免费版有固定域名
- ✅ 配置简单，图形界面
- ✅ 支持Windows/Mac/Linux

### 步骤详解

#### 1. 注册账号
访问 https://hsk.oray.com/ 注册一个免费账号

#### 2. 下载客户端
- Windows: 下载花生壳客户端安装包
- Mac: 下载 Mac 版本
- Linux: 下载对应版本

#### 3. 登录并添加映射

打开花生壳客户端，登录后：

1. 点击「添加映射」
2. 填写以下信息：
   - **应用名称**: FamilyAlbum
   - **映射类型**: HTTP
   - **内网主机**: `127.0.0.1`
   - **内网端口**: `3000`
   - **外网端口**: 系统自动分配

3. 点击「保存」

#### 4. 获取外网地址

保存后会显示一个外网地址，类似：
- `http://xxx.gicp.net`
- `http://xxx.x61.vip`

这就是你的公网访问地址！

#### 5. 分享给朋友

把这个地址发给朋友，他们就可以在任何地方访问你的相册了！

### 注意事项
- 免费版每月有1GB流量限制（家庭使用足够）
- 免费版有2个映射额度
- 电脑需要保持开机

## 方案二：cpolar

### 步骤详解

#### 1. 注册账号
访问 https://www.cpolar.com/ 注册免费账号

#### 2. 下载并安装客户端

#### 3. 认证
```bash
cpolar authtoken 你的authtoken
```
（authtoken 在官网后台可以找到）

#### 4. 启动隧道
```bash
cpolar http 3000
```

#### 5. 获取地址

启动后会显示类似：
```
https://xxxxxxxx.cpolar.cn -> http://localhost:3000
```

`https://xxxxxxxx.cpolar.cn` 就是你的公网地址！

### 注意事项
- 免费版域名每次启动会变化
- 免费版有4个隧道额度
- 国内访问速度不错

## 方案三：ngrok

### 步骤详解

#### 1. 注册账号
访问 https://ngrok.com/ 注册免费账号

#### 2. 下载客户端

#### 3. 认证
```bash
ngrok config add-authtoken 你的authtoken
```

#### 4. 启动隧道
```bash
ngrok http 3000
```

#### 5. 获取地址

启动后会显示类似：
```
Forwarding  https://xxxx-xx-xx-xxx-xx.ngrok-free.app -> http://localhost:3000
```

### 注意事项
- 免费版域名每次启动会变化
- 国内访问可能较慢
- 国际知名，稳定性好

## 🚀 完整使用流程

### 第一步：启动应用
双击 `start.bat` 启动前后端服务

### 第二步：启动内网穿透
按照上面的步骤启动你选择的内网穿透工具

### 第三步：分享地址
把获得的外网地址发给朋友

### 第四步：朋友访问
1. 朋友打开浏览器，输入你的外网地址
2. 注册/登录账号
3. 输入邀请码加入相册
4. 开始上传和查看照片！

## 💡 省电技巧

如果用自己电脑做服务器，可以：

1. **关闭显示器**：只关闭显示器，电脑主机保持运行
2. **设置电源计划**：设置电脑从不休眠
3. **使用笔记本**：笔记本电脑更省电
4. **使用旧手机**：有些内网穿透工具支持手机版

## ❓ 常见问题

### Q: 朋友访问速度慢怎么办？
A: 换用花生壳，国内速度最快

### Q: 每次重启地址都变了怎么办？
A: 花生壳免费版有固定域名，推荐使用

### Q: 流量不够用怎么办？
A: 家庭使用1GB/月通常足够，如果不够可以：
- 压缩图片后再上传
- 注册多个账号轮流使用

### Q: 电脑关机了怎么办？
A: 朋友就无法访问了，解决方案：
- 让电脑保持开机
- 使用树莓派等低功耗设备
- 考虑免费云平台方案
